# AI Tech PM 30 Days

30 天 AI 技术项目经理学习项目。

## 目录

- `Day01-LLM-base` — LLM 基础
- `Day02-LLM-chat` — 命令行连续对话（流式输出）

## 快速开始（Day02）

```bash
cd Day02-LLM-chat
python -m venv .venv
.venv\Scripts\activate        # Windows
pip install -r requirements.txt
cp .env.example .env          # 填写你的 API Key
python app.py
```
